crypto_data_fetcher

## Features

- fetch (ohlcv/funding rate) as pandas DataFrame
- incremental fetch

## Install

```bash
pip install "git+https://github.com/richmanbtc/crypto_data_fetcher.git@v0.0.17#egg=crypto_data_fetcher"
```
